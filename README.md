# نموذج دفتر تريكس

نموذج محلي لتجربة تسجيل نتائج تريكس باستخدام أوراق وأحداث الجولة بدل إدخال النقاط يدويًا.

## التشغيل

من داخل هذا المجلد:

```powershell
python -m http.server 8787 --bind 127.0.0.1
```

ثم افتح:

```text
http://127.0.0.1:8787
```

## الموجود حاليًا

- تريكس فردي وشراكة.
- شيخ الكبة، مع خيار التدبيل.
- البنات، مع توزيع كل بنت وخيار التدبيل.
- الديناري بعدّاد يحدد عدد الأوراق لكل لاعب.
- لطش بتسجيل عدد اللمّات.
- ترتيب جولة تريكس.
- تتبع الطلبات المنجزة داخل كل مملكة والانتقال بين الممالك الأربع.
- طرنيب بالشراكة، مع الطلب ونوع الطرنيب واللمّات.
- حساب الكبّوت والطلب الناجح أو الفاشل.
- اختيار نهاية مباراة طرنيب عند 31 أو 41 أو 61 نقطة.
- تريكس كومبلكس فردي وشراكة بأربع ممالك.
- جولة كومبلكس تجمع الشيخ والبنات والديناري واللطش، ثم جولة تريكس.
- هاند فردي وشراكة بإدخال نتيجة كل جولة مباشرة.
- اختيار واضح للنتيجة الموجبة أو السالبة.
- تمييز اللاعب الذي حقق هاند في سجل الجولة دون تغيير النقاط تلقائيًا.
- نهاية مرنة: يدويًا، بعد 5 جولات، أو عند وصول نتيجة إلى سالب 200.
- احتساب الفائز صاحب أقل مجموع.
- حساب تلقائي، سجل للجولات، وتراجع.
- حفظ المباراة داخل المتصفح.

هذه نسخة تجريبية، وقواعد التدبيل تحتاج مراجعة نهائية مع لاعبين قبل النشر العام.
